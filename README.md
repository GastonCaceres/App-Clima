# 🌦️ Climas de Argentina

Aplicación web desarrollada con React que permite consultar el clima actual por capital de cada provincia en Argentina, utilizando la API de OpenWeatherMap y datos geográficos de GeoNames.

##  Demo

Podés ver la aplicación funcionando acá:  
https://app-clima-flame.vercel.app/

---

##  Funcionalidades

- Selección de capitales por provincias de Argentina
- Consulta de clima actual en tiempo real
- Visualización de temperatura en grados Celsius
- Información adicional del clima (estado del tiempo)
